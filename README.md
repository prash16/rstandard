### rstandard

Calculating standard deviation is very important in any statistical analysis.

### Installation

``` r
devtools::install_github("prash16/rstandard")
```

### Quick demo

Calculating standard deviation via `standard_deviation()`:

``` r
library(rstandard)
standard_deviation(c(1,2,3,5))
#> [1] 1.47902
```
